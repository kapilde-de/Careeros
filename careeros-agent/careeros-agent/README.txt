CareerOS Agent v1.1 — Quick Start

1. Double-click "setup.bat"
2. Wait (~1-2 mins first time)
3. Enter your details
4. Add your CV in data/cv.md
5. Double-click "start.bat"

Dashboard:
http://localhost:3939

That's it.
